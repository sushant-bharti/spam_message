📩 Spam Message Detector (NLP)

Overview :
This project is a Spam Message Detection system built using natural language processing(NLP) and Machine Learning
It classifies a particular message as Spam or Not Spam by analyzing the text content. 

I have Used Naive Bayes algorithm to classify message as spam or not spam.Naive Bayes works well for text classification problems because it calculates the probability of words appearing in spam or normal messages.


Approach : 

Clean and preprocess SMS text data.
Perform Exploratory Data Analysis (EDA) to understand patterns in spam and ham messages.
Convert text into numerical vectors using TF-IDF / Bag of Words.
Train and evaluate multiple Machine Learning models.
Save the trained model using Pickle for future predictions.

Output : 
The trained model can take a message as input and predict whether it is Spam or Not Spam.


Future Work : 
Improve performance using advanced NLP models
